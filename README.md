# esp32_XBOX_SERIE_S_CONTROL_DECODE
1. SUMÁRIO
   
   O código, utilizado no arduino IDE, decodifica os controles do XBOX (série S), mais indicado para o controle de Drones e outros veículos
   por se utilizar da tecnologia BLE Bluetooth Low Energy, com baixo consumo de energia, diferente do controle do PS4 que utilizad o Bluetooth
   comum e gasta muito energia, e o alcançe, dependendo da congifuração de hardware, pode alcançar  entre 80 a 150 metros de visada direta.

   Faixa de frequência: 2.4GHz.

2. Hardware
   
   Foram utilizados no projeto os seguintes itens:
   2.1 O novo SOC ESP32-C3 - mini 1 - 32 bits - (que tem Bluetooth 5.0) - com arquitetura RISC-V - single core @160Mhz
